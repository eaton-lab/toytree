# add tip text

The ``add_tip_text`` method is used to add text labels relative to the tip node positions on a tree drawing. It can be used to add tip labels to a tree, providing a greater variety of styling options that the ``tip_labels`` argument to ``ToyTree.draw()``. Or, it can be used to add additional text annotations to a tree drawing to indicate additional information.
